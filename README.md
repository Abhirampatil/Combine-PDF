# PDF Merger using Python

This project provides a simple Python script to merge multiple PDF files into a single PDF using the `pypdf` library.

---

## 📌 Features

- Merge multiple PDF files into one
- Supports files from:
  - Same directory
  - Relative paths (folders)
  - Absolute system paths
- Lightweight and easy to use

---

## 🛠️ Requirements

- Python 3.x
- `pypdf` library

Install the required library using:

```bash
pip install pypdf
